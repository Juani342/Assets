## 📝 Descripción
Este proyecto es una aplicación web interactiva que utiliza la **Canvas API** de HTML5 para recrear una escena inspirada en el universo de **Halo**. El objetivo es demostrar el dominio de la manipulación de gráficos por medio de código (JavaScript) y la maquetación responsiva con **Bootstrap 5**.

## 🛠️ Tecnologías Utilizadas
* **HTML5 & CSS3**: Estructura y diseño con estilo HUD de la UNSC.
* **JavaScript (Vanilla)**: Renderizado de gráficos en el elemento `<canvas>`.
* **Bootstrap 5**: Framework para el diseño responsivo y componentes de UI.
* **Google Fonts**: Tipografía técnica para una mejor experiencia de usuario.
